# NecroCore
All files on the NecroCore plugin's GitHub are included in the latest version of the plugin, which is listed on Modrinth
NecroCore: Hardcore Revival System  Transform hardcore death with concussion mechanics. Players enter 5-minute savable states, become ghosts if unrescued. Revive via risky totem rituals with failure chances. Includes /necro, /ghostnow, /necroforce commands. Perfect for Paper 1.20.1+.
